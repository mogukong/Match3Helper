# 消消乐助手 (Match3Helper)

一个 Android 辅助工具 App，通过**无障碍服务 + 截屏 + 图像识别 + AI 决策**，在玩小程序消消乐时实时提示最优消除走法。

## 功能特性

- ✅ **无障碍服务截屏**：实时获取游戏画面
- ✅ **棋盘自动识别**：通过颜色采样识别 7×7 棋盘上的 5 种宝石（高性能整数 RGB 距离）
- ✅ **AI 综合决策**：
  - 普通交换：模拟所有走法，按得分排序
  - **特殊棋子组合检测**：彩虹球+彩虹球=全屏、火箭+火箭=十字、火箭+炸弹=3行3列等
  - **特殊棋子生成**：4连→火箭、5连→彩虹球
  - **道具建议**：无好走法时提示使用刷新/锤击道具
- ✅ **悬浮窗提示**：支持交换箭头、刷新提示、锤击提示三种模式
- ✅ **棋盘校准**：手动框选棋盘区域，适配不同手机分辨率

## 特殊棋子与组合效果

| 组合 | 效果 | AI 优先级 |
|------|------|----------|
| 🌈+🌈 | 全屏消除 | 🔥 最高 |
| 🌈+颜色 | 消除所有该颜色 | 🔥 极高 |
| 🚀+🚀 | 十字消除 | 🔥 高 |
| 🚀+💣 | 3行3列大消除 | 🔥 高 |
| 💣+💣 | 大范围爆炸 | 🔥 高 |
| 🌈+🚀/💣 | 全色变火箭/炸弹 | 🔥 中高 |
| 5连直线 | 生成彩虹球 | 🔥 中 |
| 4连直线 | 生成行/列火箭 | 🔥 中低 |

## 道具支持

- **刷新**：棋盘无有效走法时，AI 会提示使用刷新道具重新排列
- **锤击**：AI 会评估每个位置被敲掉后的连锁得分，提示最佳锤击点

## 使用步骤

### 1. 编译安装

用 Android Studio 打开本项目，编译并安装到手机。

### 2. 开启权限

打开 App，依次点击：

1. **「开启无障碍服务」** → 跳转到系统设置 → 找到「消消乐助手」→ 开启
2. **「校准棋盘区域」** → 按住屏幕左上角拖动到右下角，框选整个 7×7 棋盘 → 松开手指完成校准
3. **「开始辅助」** → 授权截屏权限 → 辅助开始运行

### 3. 进入游戏

打开微信小程序的消消乐游戏，保持棋盘在屏幕可见位置。App 会每 2 秒自动截屏分析一次，并在屏幕上用**绿色高亮圈 + 金色箭头**提示最优交换位置。

### 4. 停止辅助

点击「停止辅助」或从通知栏关闭。

## AI 计分策略

| 因素 | 权重 | 说明 |
|------|------|------|
| 基础消除得分 | 1.0× | 消除 3 个=300, 4 个=600, 5+ 个=1000+ |
| 连锁反应 | +300/次 | 一步引发多次掉落消除 |
| 生成特殊棋子 | +500/个 | 4+ 消除生成火箭/炸弹等 |
| 增加步数 | +400/次 | 4+ 消除奖励 +1 步，非常关键 |

## 适配说明

### 颜色阈值微调

不同手机屏幕显示颜色有差异，如果识别不准，请修改 `BoardRecognizer.kt` 中的 `COLOR_PROFILES`：

```kotlin
GemProfile(BoardState.PINK, "粉", hMin = 260f, hMax = 340f, ...)
```

根据你的截图，用取色工具测出每种宝石的 HSV 值后调整范围。

### 棋盘尺寸

当前适配 **7×7** 棋盘。如果游戏不同，修改：
- `BoardState.kt` 中的 `ROWS` 和 `COLS`
- `BoardRecognizer.kt` 中的对应常量

## ⚠️ 注意事项

1. **仅做提示，不自动点击**：本工具只显示箭头提示，不会自动操作游戏，相对安全
2. **截屏权限**：Android 系统截屏需要用户每次手动授权
3. **无障碍服务**：部分手机需要关闭电池优化，否则后台可能被杀死
4. **合规提醒**：请勿用于自动代打或外挂性质操作，仅作学习参考
